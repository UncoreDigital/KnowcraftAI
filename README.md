# FintechBrain

A full-stack fintech application built with React, TypeScript, and Express.

## How to Run

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run build
npm run dev
```

### Production
```bash
npm run build
npm start
```

The application will be available at `http://localhost:5000`

## Features
- Chat interface
- Analytics dashboard
- Audit logs
- Knowledge base management
